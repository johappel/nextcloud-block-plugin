# Nextcloud Block Plugin

Integrate Nextcloud directories as blocks in WordPress and display them as a tree structure.

## Tree
```markdown
nextcloud-block-plugin/
│
├── build/                # Compiled assets
│   └── ...               # (Generated by the build process)
│
├── node_modules/         # Node.js packages
│   └── ...               # (Installed via npm)
│
├── src/                  # Source files for the plugin
│   ├── block/            # Source files for the Gutenberg block
│   │   ├── block.json    # Block metadata
│   │   ├── edit.js       # Editor script
│   │   ├── editor.scss   # Editor styles
│   │   ├── index.js      # Block registration
│   │   ├── save.js       # Save function
│   │   ├── style.scss    # Frontend and editor styles
│   │   └── view.js       # Script for the frontend view
│   └── ...
│
├── .editorconfig         # Coding style definitions
├── .gitignore            # Specifies intentionally untracked files to ignore
├── nextcloud-block-plugin.php  # Main plugin file
├── package.json          # Node.js manifest
├── package-lock.json     # Node.js lock file
└── readme.txt            # Readme file for the WordPress plugin repository
```
